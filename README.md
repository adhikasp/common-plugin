Spring Boot Common Plugin
--------------------------

Add this dependency in your ```pom.xml```

```xml
<dependencies>
  ...
  <dependency>
   <groupId>com.gdn.spring.boot.plugin</groupId>
   <artifactId>common-plugin</artifactId>
   <version>0.0.1-RELEASE</version>
 </dependency>
 ...
</dependencies>
```
